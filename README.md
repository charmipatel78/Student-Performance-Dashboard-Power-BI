# 🎓 Student Performance Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive **Student Performance Dashboard** built using Power BI.
It helps analyze student data such as **scores, attendance, and behavior** through clear and dynamic visualizations.

---

```
 └── Students.xlsx

📂 Main File
 └── Student Performance.pbix

README.md
```

---

## 🚀 Features

* 📌 Overview of student performance
* 📈 Interactive charts (Bar, Line, Area)
* 📊 Subject-wise score analysis
* 📅 Attendance tracking
* 🧠 Behavior insights
* 🔍 Student detail drill-down page

## 📌 How to Use

1. Download the `.pbix` file from the **Main File** folder
2. Open it in Power BI Desktop
3. Explore dashboards using filters and visuals

* Attendance data
* Behavior data
* Student scores
* Student details

## 💡 Insights

This dashboard helps in:

* Identifying top and low-performing students
* Tracking attendance patterns
* Understanding behavior impact on performance
